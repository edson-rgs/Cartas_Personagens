# 🃏 Cartas de Personagens

Este projeto foi desenvolvido como parte dos meus estudos em **JavaScript**, com foco em **Programação Orientada a Objetos (POO)**.

A proposta consiste na criação de um sistema de **cartas de personagens**, onde cada carta possui atributos e comportamentos próprios. O projeto aplica, na prática, os principais conceitos de POO, contribuindo para a construção de uma base sólida em desenvolvimento orientado a objetos.

---

## 📚 Conceitos Aplicados

Durante o desenvolvimento, foram explorados os principais pilares da Programação Orientada a Objetos em JavaScript:

### 🔹 Classes e Objetos
Estruturação do código utilizando classes para representar personagens e objetos instanciados como cartas.

### 🔹 Encapsulamento
Organização dos atributos e métodos, garantindo maior controle, segurança e clareza no código.

### 🔹 Herança
Criação de classes específicas como `Mago`, `Arqueiro` e `ArqueiroMago`, que herdam características de uma classe base `Personagem`.

### 🔹 Polimorfismo
Sobrescrita de métodos nas subclasses, permitindo comportamentos distintos para cada tipo de personagem, como ataques especiais.

### 🔹 Abstração
Definição de atributos e comportamentos comuns, facilitando a reutilização e manutenção do código.
